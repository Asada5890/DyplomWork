# Инструкция по запуску проекта

Для корректной работы вашего проекта с использованием FastAPI, MongoDB, Redis и SQLite, следуйте этим шагам.

## 1. Установка зависимостей

Для начала необходимо установить все необходимые зависимости для вашего проекта.

### Windows
1. Скачайте и установите [Python](https://www.python.org/downloads/) (если еще не установлен).
2. Скачайте проект и перейдите в директорию проекта через командную строку.
3. Установите все зависимости, используя команду:

   ```bash
   pip install -r requirements.txt
   ```

## 2. Запуск MongoDB

Приложение использует MongoDB для хранения информации для строительных материалов. Чтобы запустить MongoDB, выполните следующие шаги:

### 2.1 Запуск через MongoDB Compass

MongoDB Compass — это графический интерфейс для работы с MongoDB. Чтобы запустить MongoDB через Compass:

1. Скачайте и установите [MongoDB Compass](https://www.mongodb.com/products/compass).

2. Запустите MongoDB Compass и подключитесь к локальной базе данных MongoDB, используя стандартный URI подключения:

   ```
   mongodb://localhost:27017
   ```

3. После подключения убедитесь, что ваша база данных настроена правильно.

### 2.2 Запуск MongoDB через Docker (если не используете Compass)

Если вы используете Docker, вы можете запустить MongoDB с помощью следующей команды:

```bash
docker run --name mongodb -d -p 27017:27017 mongo:latest
```

Это создаст контейнер с MongoDB, который будет доступен на порту 27017.

## 3. Запуск Redis

Для работы с кешированием и сессиями приложение использует Redis. Чтобы запустить Redis:

### 3.1 Запуск через Docker

Если у вас установлен Docker, вы можете запустить Redis с помощью следующей команды:

```bash
docker run --name redis -d -p 6379:6379 redis:latest
```

Это создаст контейнер с Redis, который будет доступен на порту 6379.

### 3.2 Запуск через установку

1. Установите Redis на ваш компьютер, следуя инструкциям на официальном сайте: [Redis Downloads](https://redis.io/download).
2. Запустите Redis с помощью команды (в зависимости от вашей операционной системы):

   ```bash
   redis-server
   ```

## 4. Запуск SQLite

SQLite используется в проекте для работы с базой данных. SQLite является серверной базой данных, которая хранит данные в одном файле. Для работы с ней не требуется дополнительных настроек.

Просто убедитесь, что файл базы данных `db.sqlite3` существует в проекте и доступен для чтения/записи.

## 5. Запуск приложения

Теперь, когда все сервисы настроены, вы можете запустить приложение.

### Запуск FastAPI

1. Для запуска приложения с использованием FastAPI, выполните команду:
    #### Windows
   ```bash
   python main.py
   ```
    ####MacOS
    ```bash
   python3 main.py
   ```

2. Перейдите по следующему адресу в браузере:

   ```
   http://localhost:8000
   ```

3. Если приложение запущено правильно, вы увидите интерфейс FastAPI и сможете использовать все эндпоинты приложения.

## 6. Проверка работы

Для проверки работы приложения, сделайте следующие шаги:

1. Перейдите в интерфейс FastAPI по адресу `http://localhost:8000`.
2. Включите все необходимые эндпоинты для проверки функциональности.
3. Убедитесь, что ваше приложение правильно работает с MongoDB, Redis и SQLite.

## 7. Завершение

Теперь вы можете использовать ваше приложение! Убедитесь, что все сервисы (MongoDB, Redis и SQLite) работают и настроены правильно.

Если возникнут какие-либо проблемы, проверьте логи FastAPI для получения дополнительной информации.

---

> **Примечание**: Если вы используете другие операционные системы или инструменты для установки MongoDB или Redis, следуйте соответствующим инструкциям в их документации.
